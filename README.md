# 📝 To-Do List Website

A simple to-do list web app built with plain **HTML, CSS, and JavaScript**. No server, no setup — it runs directly in the browser and works perfectly on **GitHub Pages**.

## Features

- Add tasks
- Click a task (or its circle) to mark it done / not done
- Delete tasks
- Tasks are **saved in your browser** (using localStorage), so they stay even after you refresh or close the page
- Clean, responsive design

## Project Structure

```
todo-website/
├── index.html      # The entire app (HTML + CSS + JavaScript)
├── README.md
└── .gitignore
```

## Run It Locally

Just open `index.html` by double-clicking it — it opens in your browser. That's it.

## Host It Online with GitHub Pages (Free Live Link)

1. Push this folder to a GitHub repository.
2. On GitHub, go to your repo → **Settings** → **Pages** (left sidebar).
3. Under **Source**, choose the **main** branch and the **/(root)** folder, then click **Save**.
4. Wait about a minute. GitHub will give you a live link like:
   `https://<your-username>.github.io/<repo-name>/`
5. Open that link — your to-do app is now live on the internet! 🎉

> Important: The file **must** be named `index.html` for GitHub Pages to load it automatically.
